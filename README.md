# Brainstore

An AI learning platform with a 3D brain interface.

## Tech Stack

- React (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)
- Claude AI (Anthropic)

## Setup

1. Install dependencies:
   ```bash
   npm run install-all
   ```

2. Configure environment variables in `server/.env` and `client/.env`

3. Start development servers:
   ```bash
   npm run dev
   ```
